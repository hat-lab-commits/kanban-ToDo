# Kanban ToDo App

シンプルなカンバン形式のToDoアプリです。

##  使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）
- Git / GitHub

##  実装機能
- タスク追加
- 優先順位設定（高・中・低）
- チェック機能（完了表示）
- カラム移動（ドラッグ＆ドロップ）
- ステータス管理（未着手 / 進行中 / 完了）

##  工夫した点
- データ構造を意識して設計
- render関数による再描画処理
- DOM操作を用いた状態管理
- イベントリスナーによるドラッグ実装
